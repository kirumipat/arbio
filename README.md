# Arbio
## This script install ArchLinux

### Minamal install
  - pacman -Sy git
  - git clone https://github.com/kirumipat/arbio
  - cd arbio
  - chmod +x arbiostart_mini.sh
  - ./arbiostart_mini.sh

  - pacman -Sy git
  - git clone https://github.com/kirumipat/arbio
